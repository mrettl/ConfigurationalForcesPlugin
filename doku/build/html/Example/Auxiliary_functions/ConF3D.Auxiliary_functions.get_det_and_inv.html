ConF3D.Auxiliary_functions.get_det_and_inv¶

ConF3D.Auxiliary_functions.get_det_and_inv(jac)[source]¶

Inverts a symbolic matrix and calculates the determinant.

This functions works for 3d-elemnts and plane strain 2d-elements. If the determinant of the 3x3 matrix is zero, the 2d- plane strain case is assumed.

Parameters
jac(2 or 3,2 or 3) SymPy matrix

Symbolic 3x3 matrix to be inverted

Returns
inv_jac(2 or 3,2 or 3) SymPy matrix

Inverted symbolic matrix

detSymPy expression

Determinant of the matrix
