ConF3D.Auxiliary_functions.lambdify_numba¶

ConF3D.Auxiliary_functions.lambdify_numba(args, arg_names, expr, expr_name='_lambdified')[source]¶

Generates Numba code for a given symbolic expression.

This function is called by gen_Integration_Numba This function isn’t tested and is an alternative to the function which generates C-code.
