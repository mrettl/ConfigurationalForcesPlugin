ConF3D.Auxiliary_functions.gen_2D_Ten_from_vec¶

ConF3D.Auxiliary_functions.gen_2D_Ten_from_vec(Vec)[source]¶

Create a Second order Tensor from vector notation.

This function is Abaqus specific. eg. vector notation of the stress tensor: (S11,S22,S33,S12,S13,S23)

Parameters
Vec(6) nd-array consisting of SymPy symbols or SymPy matrix

Second order tensor in vector notation

Returns
S_TenSymPy matrix
