ConF3D.Auxiliary_functions.inversion_and_det_3x3¶

ConF3D.Auxiliary_functions.inversion_and_det_3x3(m_in)[source]¶

Inverts a symbolic 3x3 matrix and calculates the determinant. This can be also accomplished with SymPy, the use of this function is for performance reasons only.

Parameters
m_in(3,3) SymPy matrix

Symbolic 3x3 matrix to be inverted

Returns
minv(3,3) SymPy matrix

Inverted symbolic matrix

determinantSymPy expression

Determinant of the matrix
