ConF3D.Auxiliary_functions.gen_Integration_C_static¶

ConF3D.Auxiliary_functions.gen_Integration_C_static(rst, coord, Element_U, S, PENER, SENER, int_points, weights, expr, expr_name)[source]¶

Generates C99 code for the static implementation of configurational forces at element nodal position.

This function is called by gen_Configurational_Forces_Static() and performs the numeric integration of function values determined at the integration points. Have a look at this function too see how rst, coord, Element_U, S, SENER, PENER are defined. The numeric representation at the integration point is generated by lambdify_C().

Parameters

Element_U(Number of Nodes,3) SymPy matrix

Symbolic expression for which the numerical implementation should be generated
